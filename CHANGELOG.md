## [v1.0.0] - 2023-12-06

### Added
- Initial release of the backup utility
- Automated backup functionality for Tally-like systems
- ZIP compression for backup files
- Automatic cleanup of old backups
- Comprehensive logging system
- Error handling and reporting
- Configurable parameters
- Detailed documentation

### Features
- Creates timestamped backup folders
- Compresses backups to ZIP archives
- Manages backup retention (keeps specified number of backups)
- Generates detailed logs of backup operations
- Handles errors gracefully with appropriate messages
- Configurable source, target, and retention settings
